Introduction

For the pairs with ECT FDR < 0.1, we performed colocalization analysis on the supporting SNPs. We considered a ±250 kb window around each SNP. Within this region, we conducted association tests for the factor and the SNPs. Finally, we calculated the colocalization posterior probability (PP4) using coloc package.
