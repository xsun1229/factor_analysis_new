Introduction

For the pairs with ECT p-value < 0.0005, we performed colocalization analysis on the supporting SNPs. We considered a ±250 kb window around each SNP. Within this region, we conducted association tests for the factor and the SNPs. Finally, we calculated the colocalization posterior probability (PP4) using coloc package.

For pairs with ECT p-values greater than 0.0005, we sampled a control set consisting of five times as many pairs as those with ECT p-values below 0.0005 and performed colocalization analysis.
